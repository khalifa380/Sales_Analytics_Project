# Sales_Analytics_Project
This project analyzes sales data using Python (Jupyter Notebook) and Power BI to create an interactive dashboard. It highlights key business insights such as sales trends, top products, and customer performance. The project demonstrates skills in data analysis, visualization, and transforming raw data into actionable insights for decision-making.

Sales Analytics Dashboard
1️ Project Overview

This project analyzes sales data using Python (Jupyter Notebook) and Power BI to create an interactive dashboard. The goal is to convert raw sales data into meaningful insights that help businesses understand product performance, customer trends, and sales patterns.

2️ Project Objectives

Analyze sales data to discover useful business insights

Build an interactive Power BI dashboard

Identify top-performing products and customers

Visualize sales trends over time

3️ Technologies Used

Python

Jupyter Notebook

Pandas

Power BI

Data Visualization

7️ Key Insights

Some product categories generate higher revenue than others

A small group of customers contributes significantly to sales

Sales patterns vary across time periods

Open the Jupyter Notebook to see data analysis.

Open the Power BI file (.pbix) to explore the dashboard.

Use slicers and filters to interact with the data.
